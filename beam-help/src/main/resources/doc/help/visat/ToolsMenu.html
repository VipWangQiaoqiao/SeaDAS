Tools Menu

VISAT's Tools Menu

The following tools all generate new products or nodes of a product from the currently selected input product(s). To view their created raster datasets after processing is complete, change to the product browser, expand the new product's nodes and double click on a band node to open an image view. Note that you really get new products which can be saved and opened again for later use.

  • Create Band by Band Maths
  • Create DEM-related Bands
  • Create NRCS Bands (ASAR)
  • Create Filtered Band
  • Create Geometry Container
  • Reprojection
  • Orthorectification
  • Mosaic
  • Data Flip

The following tools manage the geo-coding associated with a product.

  • Attach Pixel Geo-Coding
  • Detach Pixel Geo-Coding

VISAT Integration of Scientific Data Processors

The following scientific data processors are also provided as stand-alone applications to be run either from the command-line in batch mode or interactive mode. Of course, within VISAT they are exclusively run in interactive mode.

  • SMAC Processor(MERIS/(A)ATSR)
  • Smile Correction Processor(MERIS)
  • Cloud Probility Processor(MERIS)
  • Radiance-To-Reflectance Processor(MERIS)
  • SST Processor((A)ATSR)
  • FLH/MCI Processor
  • Level 3 Mosaic Processor
  • Level-3 Binning Processor
  • NDVI Processor(MERIS)
  • Collocation
  • Image Analysis
    • Spectral Unmixing
    • EM Cluster Analysis
    • K-Means Cluster Analysis
