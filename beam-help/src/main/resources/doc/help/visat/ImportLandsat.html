Import Landsat

The Landsat reader enables VISAT to import Landsat TM data products in either the Fast format (Landsat5) or in GeoTIFF (Landsat4,5) format. The instruments ETM+ (Landsat7), OLI, TIRS (Landsat8) are only support in GeoTIFF format. MSS data from Landsat 1 to 5 is also only supported in GeoTIFF format.

More about Landsat characteristics can be found at http://landsat.usgs.gov/resources/project_documentation.php.
