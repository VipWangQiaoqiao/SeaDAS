The 'About VISAT' Window

Selecting the About... entry in the Help Menu brings up a window similar to the following image

Credits Button
Clicking on this button brings up a window with a list of people, organisations and companies we want to thank for their contributions to develop this software. You can also find a list of the members of the BEAM Development Team.
System Info Button

This brings up a window which lists software and OS-specific properties in a table. You can mark entries with the mouse and copy the properties by using CTRL + C.
