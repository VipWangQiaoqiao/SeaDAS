No-Data Overlay

   No-Data Overlay

Toggles the overlay of the no-data mask within a band's Image View. The overlay style properties can be modified using the no-nata overlay page in the Preferences Dialog.

The no-data mask overlays displays all the band's pixels with undefined sample values. In order to control no-data pixels, use the band's no-data value and/or the band's valid pixel expression in the Property Editor for the selected band.
