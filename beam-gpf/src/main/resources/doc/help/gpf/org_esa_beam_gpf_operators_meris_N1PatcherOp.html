Meris.N1Patcher Operator

Meris.N1Patcher Operator Description

Overview

Name: Meris.N1Patcher
Full name: org.esa.beam.gpf.operators.meris.N1PatcherOp
Purpose: Copies an existing N1 file and replaces the data for the radiance bands
Version: 1.1

Description

The N1PatcherOp copies an existing N1 file and replaces the data for the radiance band with the data from the given product

Configuration Elements:

  • originalFilePath the path of the N1 file that should be patched
  • patchFilePath the path for the patched file

Sources

Name Description
n1Product The N1 file which is used as a template.
sourceProduct The source product provides the data to be written into the patched file.

Parameters

Name Data Type Default Description Constraints
patchedFile File The file to which the patched L1b product is written. not null; non empty
