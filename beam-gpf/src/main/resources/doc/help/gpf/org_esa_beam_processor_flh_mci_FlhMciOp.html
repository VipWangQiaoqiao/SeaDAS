FlhMci Operator

FlhMci Operator Description

Overview

Name:FlhMci
Full name:org.esa.beam.processor.flh_mci.FlhMciOp
Purpose:Computes fluorescence line height (FLH) or maximum chlorophyll index (MCI).
Version:2.0

Description

An operator for computing fluorescence line height (FLH) or maximum chlorophyll index (MCI).

Sources

Name Description
sourceProduct

Parameters

Name Data Type Default Description Constraints

upperBaselineBandName String The name of the upper wavelength band defining the baseline
signalBandName String The name of the signal band, i.e. the band for which the baseline height is calculated
lineHeightBandName String The name of the line height band in the target product
slope boolean true Activates or deactivates calculating the slope parameter
slopeBandName String The name of the slope band in the target product
maskExpression String A ROI-mask expression used to identify pixels of interest
cloudCorrectionFactor float 1.005 The cloud correction factor used during calculation
invalidFlhMciValue float NaN Value used to fill invalid FLH/MCI pixels
